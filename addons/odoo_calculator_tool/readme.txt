# Odoo Systray Calculator Pro

A lightweight **calculator tool** integrated directly into the Odoo systray.
Easily perform quick calculations without leaving your workflow.

---

## ✨ Features

* 🖩 **Quick Access**: Open the calculator from the systray icon.
* 🎯 **Draggable UI**: Move the calculator around the screen using the drag handle.
* ❌ **Close Button**: Hide the calculator instantly with one click.
* 📜 **Calculation History**: Scrollable & selectable history for copy-paste.
* ➕ **Basic Operations**: Add, subtract, multiply, divide, toggle sign, and decimals.
* 🔒 **Safe Input**: Read-only display field to avoid accidental overwriting.
* ⚡ **Modern UI**: Styled with SCSS, lightweight, and responsive.

---

## 🚀 Installation

1. Copy this module into your Odoo `addons` folder:

   ```
   odoo_systray_calculator_pro/
   ```
2. Update the Odoo app list:

   ```
   ./odoo-bin -u odoo_systray_calculator_pro -d <your_database>
   ```
3. Activate the module in **Apps > Calculator Pro**.

---

## 📚 Usage

* Click the **calculator icon** in the top systray.
* Use the on-screen buttons to perform calculations.
* Drag the calculator around using the **bars icon**.
* Close it with the **× button**.

---

## 🛠 Compatibility

* ✅ Odoo 19.0 (tested)
* ✅ Python 3.10+
* Works with **Enterprise** and **Community** editions.
